Full-Stack Job Tracking Application

Project Description

Developed a full-stack job tracking application to help users monitor job applications, statuses, and notes. The app supports user authentication, job CRUD, and filtering by status. Frontend built in React with responsive pages; backend is a RESTful API built with Node and Express and PostgreSQL for persistence.

Skills & Technologies

  • React.js, Node.js, Express.js
  • PostgreSQL
  • REST API design, Authentication
  • Responsive UI, CSS grid/flexbox
