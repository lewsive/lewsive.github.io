Mini-Compiler (Team Lead)

Project Description

Led a 4-member team to build a mini-compiler in C that converts assembly-level code into machine instructions. Implemented lexical analysis with FLEX, syntax parsing with BISON, and key components for error reporting, symbol table management, and code generation. This project strengthened my understanding of language processing and systems-level programming.

Skills & Technologies

  • C, FLEX, BISON
  • Compiler design fundamentals: lexer, parser, AST, code generation
  • Team leadership, task scheduling
